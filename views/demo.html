What's in your VidCode?

  • Objects
  • movie
  • movie is an object that uses the canvas element in HTML5 to play your video.
  • effects
  • effects is an object within the seriously library. It contains all the effects. By typing seriously.effect('blur'), we are pulling the “blur” effect out of the library.
  • seriously
  • seriously is a library which is represented by this object. By typing seriously.effect('blur'), we are pulling the “blur” effect out of the library. There is a lot of code in that library! You just cant see it ;)
  • filmgrain
  • film grain is an object within the effects object. It’s accessed via a "." dot notation.
  • Functions
  • A function is a set of arguments or intructions to run on a block of code.
  • showEffect
  • The "showeffect" function consists of the arguments listed within the function’s brackets { }. In the same way as you are your body which consists of many processes.
  • Methods
  • A method is an argument (similar to a function) which is applied to a object using a "."
  • play
  • is method which effects the "movie" object
  • interval
  • is a method which effects the "movie" object and it has two "parameters"
  • Parameters
  • A parameter is "value" which is passed from one function to another.
  • Values
  • A "value" in javascript can be any of the following: -number -true or false -"string"(i.e. a word in quotes)
