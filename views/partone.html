Objects...

A object in javascript can be anything!

In the case of our VidCode these words, such as “movie”, are containers to allow instructions to be applied to the “movie”.

We will cover this again and again! For now just get used to the word and where objects are placed within the code.

Your movie!

The word "movie" is an object that uses the canvas element in HTML5 to play your video.

Remember, an object is just a word that contains data under the hood.

With code, and computers, there are many things happening "behind the scenes" which we don't see.

So many effects!

The word "effects" is an object that contains ALL the effects.

You have three effects in your demo.

These live in a hidden "list" within the "effects" object.

Functions are actions

Functions are a HUGE piece of the computer programming pie.

A function is a set of arguments or intructions to run on a block of code.

"play" is a function...

...but it's called a method.

It tells the movie object to play. It's called a method, not a function, because it is applied directly to a object via the dot "." thing.

"interval" is also a method

It tells the movie object how many times a second to be processed.

The number "1000" is in milliseconds. So it is processing your video 1,000 times a second! That's a lot of work!
